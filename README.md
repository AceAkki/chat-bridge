# Chat-Bridge

A simple web application that allows users to send messages, call, text, or copy links using deep links for popular messaging platforms.

## Features

- Send **custom messages** or predefined messages like `"Hi"`.  
- **Call** or **text** contacts directly.  
- Copy a **link** to the clipboard.  
- Supports deep links for:
  - **WhatsApp**
  - **Viber**
  - **Signal**
  - **Telegram**
