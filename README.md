# Chat-Bridge

**Chat-Bridge** is a lightweight, browser-based web app that lets you instantly connect with anyone across multiple messaging platforms. Whether you want to send a custom message, make a call, send a text, or share a link, Chat-Bridge generates deep links for **WhatsApp, Viber, Signal, and Telegram** — all in one place.  

Key advantages:  
- **Send messages instantly** without saving contacts.  
- **Call or text** directly from your browser.  
- **Copy links** for quick sharing.  
- **Automatic country code detection** using location API — pre-fills your number for convenience (requires location access).  
- Fully **client-side**, no server setup required — works in any modern browser.  

---

## Features

- Send **custom messages** or predefined messages like `"Hi"`.  
- Call or text contacts directly.  
- Copy a **link** to the clipboard.  
- Supports deep links for:
  - **WhatsApp**
  - **Viber**
  - **Signal**
  - **Telegram**
- Automatic country code detection for phone numbers.
